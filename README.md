This Calculator Project is a modern, interactive web-based application developed using HTML, CSS, and JavaScript. It is designed to perform both basic and scientific mathematical operations with a clean, user-friendly interface.

The calculator supports standard arithmetic operations such as addition, subtraction, multiplication, and division, along with advanced features like square (x²), power (xʸ), and square root (√). JavaScript is used to handle all calculations and user interactions, ensuring fast and accurate results.
